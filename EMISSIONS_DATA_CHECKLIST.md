This checklist covers ALL data required for comprehensive GHG Protocol and regulatory compliance reporting (including California SB253, EPA, SEC climate disclosure).

## üî• 4. SCOPE 1: DIRECT EMISSIONS (Required)

### Stationary Combustion
For each fuel type used:
- [ ] **Fuel Type** (natural gas, diesel, propane, coal, biomass, etc.)
- [ ] **Fuel Amount** (quantity consumed)
- [ ] **Fuel Unit** (therms, gallons, liters, cubic meters, tons, etc.)
- [ ] **Equipment Type** (boiler, furnace, generator, heater)
- [ ] **Equipment Location** (facility ID)
- [ ] **Emission Factor Source** (EPA, DEFRA, custom)
- [ ] **Emission Factor Value** (kg CO2e per unit)
- [ ] **GHG Breakdown** (CO2, CH4, N2O amounts if known)

### Mobile Combustion
For each vehicle type/fuel:
- [ ] **Vehicle Type** (truck, van, car, forklift, etc.)
- [ ] **Fleet Size** (number of vehicles)
- [ ] **Fuel Type** (gasoline, diesel, CNG, etc.)
- [ ] **Fuel Amount** (quantity consumed)
- [ ] **Fuel Unit** (gallons, liters)
- [ ] **Distance Traveled** (miles or km, if available)
- [ ] **Ownership Type** (owned, leased, company-controlled)

### Fugitive Emissions
For each source:
- [ ] **Source Type** (refrigeration, AC, fire suppression, manufacturing process)
- [ ] **Refrigerant/Gas Type** (R-134a, R-410A, HFC-134a, SF6, NF3, etc.)
- [ ] **Equipment ID** (unique identifier)
- [ ] **Gas Amount Leaked/Emitted** (kg or lbs)
- [ ] **Global Warming Potential (GWP)** (from IPCC AR5 or AR6)
- [ ] **Leak Detection Date** (if applicable)
- [ ] **Repair Date** (if applicable)

### Process Emissions
For each industrial process:
- [ ] **Process Type** (cement production, chemical reaction, metal smelting, semiconductor etching, etc.)
- [ ] **Process Description** (what happens)
- [ ] **GHG Gas Type** (CO2, CH4, N2O, HFCs, PFCs, SF6, NF3)
- [ ] **Gas Amount Emitted** (kg or metric tons)
- [ ] **Production Volume** (tied to emissions)
- [ ] **Abatement System** (if any: scrubbers, destruction efficiency %)
- [ ] **Emission Factor** (per unit of production)

### Agricultural Emissions (if applicable)
- [ ] **Livestock Type** (cattle, pigs, poultry, etc.)
- [ ] **Number of Animals** (average annual)
- [ ] **Enteric Fermentation** (CH4 emissions)
- [ ] **Manure Management** (CH4 and N2O)
- [ ] **Cropland Area** (hectares)
- [ ] **Fertilizer Use** (type, amount, N2O emissions)

## üí° DATA ENTRY TIPS

1. **Start with what you have**: Begin with utility bills and work outward
2. **Estimate missing data**: Use industry averages initially, improve over time
3. **Document assumptions**: Always note how estimates were made
4. **Track data sources**: Keep audit trail of where each number came from
5. **Use consistent units**: Convert everything to metric tons CO2e
6. **Validate totals**: Cross-check against spend data, production volumes
7. **Update regularly**: Set up monthly/quarterly data collection processes
8. **Automate where possible**: Integrate with existing business systems
